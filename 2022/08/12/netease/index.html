Windows端方法

注意：Windows 7 如无法执行则需升级 Powershell 到 3.0 以上，XP 不支持

第一步，版本问题

版本太高的话可能会失效导致代理成功也无法连上网络，我选择的是以下图片的版本，也可以按我提供的链接去下载 https://mydown.yesky.com/pcsoft/35385808/versions/ ，注意把自动更新换成有新版本时提醒我

第二步，安装代理

以 管理员身份 打开 Powershell，Windows 10/11 快捷入口：Win + X - Windows Powershell(管理员)(A)

复制以下代码，右键粘贴到命令行回车，打开安装菜单。

  • 随后选择 1 即安装。
  • 安装完毕后选择 3 运行。
  • 如需添加开机自启，则执行 7。
  • 最后输入 0 退出。

第三步，设置代理

打开网易云音乐客户端，进入设置页面，设置自定义代理

  • 自定义代理 ：填写服务器地址和端口号
  • 代理服务器地址：127.0.0.1 （推荐本机搭建，速度快）
  • 代理服务器端口：6666

如使用一段时间后无法解锁，则需要重新执行命令，选择 5 更新。

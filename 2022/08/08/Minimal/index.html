一、CentOS 7 Minimal网络配置

1.输入命令cd /etc/sysconfig/network-scripts进入网络配置文件目录

2.输入命令vi ifcfg-ens32，对文件进行编辑

3.输入:wq，保存退出

4.输入service network restart，重启网络服务

5.输入ping baidu.com，查看是否配置成功

二、ifconfig命令command not found解决办法

安装完centOS 7 执行ifconfig命令提示command not found

解决办法：

1、查找环境标量是否包含ifconfig。如下图，显示环境变量没有ifconfig。

2、执行 yum search ifconfig ，查找ifconfg脚本所在的本地安装包。

显示如下：

3、说明ifconfig脚本的安装包在net-tools.x86_64的安装包中，执行 yum -y install net-tools.x86_64 命令（-y命令执行过程中跳过确认），重新安装即可。

4、检查安装是否成功

5、查找ifconfig脚本的位置，执行find / -name ifconfig -print
